## CPU Components

### What You Need to Know
1. What the Control Unit (CU) is and three things it does
2. What the Arithmetic Logic Unit (ALU) is and three things it does
3. What the Memory Address Register (MAR) is and what it does
4. What the Program Counter (PC) is and what it does
5. What the Memory Data Register (MDR) is and what it does
6. What the Accumulator (ACC) is and what it does

---

### Control Unit (CU)
- Controls the execution of instructions
- Decodes instructions
- Controls timing using signals from the system clock
- Sends control signals to other components

---

### Arithmetic Logic Unit (ALU)
- Performs arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Modulus
  - Quotient division

- Performs logical operations:
  - AND
  - OR
  - NOT

- Performs binary shift operations:
  - Shifts bits left or right

---

### Special Purpose Registers

A register is a small, very fast memory location inside the CPU used to temporarily store data, instructions, or addresses currently in use.

#### Memory Address Register (MAR)
- Holds the address of the data or instruction to be fetched from or written to memory

#### Memory Data Register (MDR)
- Holds the data or instruction being transferred to or from memory

#### Program Counter (PC)
- Holds the address of the next instruction to be fetched
- Automatically increments after each instruction is fetched

#### Accumulator (ACC)
- Holds the result of arithmetic and logical operations performed by the ALU

---

### Clock Speed

- The **clock speed** is the speed at which a processor operates
- Measured in **hertz (Hz)** — the number of clock cycles per second
- 1 hertz = 1 cycle per second
- Modern processors are measured in **gigahertz (GHz)** — one billion cycles per second
- Typical PC clock speeds range between **2GHz and 4GHz**
- Everything the processor does happens on a 'tick' of the clock
- A faster clock means more instructions are fetched, decoded and executed each second

---

### Clock Speed and Performance

- In theory, a 4GHz processor operates **twice as fast** as a 2GHz processor
- In practice, many other components affect overall speed
- A slow component elsewhere in the system creates a **bottleneck**
- A bottleneck limits how quickly the whole system can operate
- The other main factors affecting CPU performance are:
  - Cache size
  - Number of cores

## CPU Performance — Cache Memory

### What You Need to Know

1. What cache memory is
2. Cache memory is located in the CPU
3. How cache acts as an intermediary between RAM and the CPU
4. Why cache is faster but more expensive than RAM
5. How the amount of cache affects CPU performance
6. The difference in size between RAM and cache memory

---

### Cache Memory

- **RAM (Random Access Memory)** is the main memory of a computer
- Reading from and writing to RAM is much slower than the speed at which the processor works
- This speed difference is a **bottleneck** that limits CPU performance
- Using faster memory technologies would solve this but increases cost
- The compromise solution is **cache memory**
- Cache is a small amount of very fast (but expensive) memory
- It stores frequently used instructions and data temporarily
- Cache acts as an **intermediary** between RAM and the CPU
- Data in cache can be accessed much more quickly than data in RAM
- If the required data is not in cache, it must be fetched from RAM
- The more cache a computer has, the more data can be held ready for the CPU
- More cache = better CPU performance

---

### Cache Size and RAM — Comparison

- A typical PC might have **8GB or 16GB of RAM**
- The same PC might have only **2MB of cache** inside the CPU
- A CPU will have “Level 1 cache” only 2kb which is extremely fast
- A CPU will have “Level 2 cache” 2mb which is fast 
- Some CPUs will have “Level 3 cache”

### Misconceptions

- misconception: Cache and RAM are the same thing
  reality: Cache is a separate, smaller, faster and more expensive memory type. RAM is larger, slower and cheaper. They serve different roles.
  trigger: When a student uses cache and RAM interchangeably
  correction_prompt: Cache and RAM are not the same. Cache is a small, very fast memory used to store frequently used data close to the CPU. RAM is much larger but slower. If data isn't in cache, the CPU has to fetch it from RAM, which takes longer.

- misconception: More RAM means more cache
  reality: RAM and cache are separate. Increasing RAM does not increase cache size.
  trigger: When a student suggests upgrading RAM to improve cache performance
  correction_prompt: RAM and cache are separate types of memory. Upgrading RAM does not increase cache. Cache size is fixed by the processor hardware.
