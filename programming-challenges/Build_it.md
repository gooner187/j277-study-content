# Build It — J277 Programming Projects

## Project 1: Basic Calculator
**Difficulty:** easy
**Description:** A text-based calculator that handles the four operations with a user menu and error checking.
**Features:**
- User menu (add, subtract, multiply, divide, quit)
- Takes two number inputs
- Performs the chosen operation
- Displays the result
- Loops back to the menu after each calculation
- Error checking for division by zero
**Constructs:** variables, casting, selection, while loops, functions, procedures

## Project 2: Penalty Shootout Game
**Difficulty:** medium
**Description:** A best-of-5 penalty shootout against the computer with sudden death and score tracking.
**Features:**
- User chooses where to shoot (left, centre, right)
- Computer randomly chooses where to dive
- Best of 5 penalties
- Sudden death if scores are tied after 5
- Score tracking and display after every kick
- Declare winner at the end
**Constructs:** variables, constants, lists, selection, for loops, while loops, random, procedures, functions, concatenation, parameters, returning

## Project 3: Number Guessing Game
**Difficulty:** easy
**Description:** Guess a secret random number with limited attempts and high/low hints.
**Features:**
- Random secret number generated
- User gets limited guesses (e.g. 5)
- Hints: "too high" or "too low"
- Displays number of guesses used
- Easy / medium / hard difficulty modes (change the range)
- Replay option
**Constructs:** random, while loops, selection, casting, functions, constants

## Project 4: Bank Account Simulator
**Difficulty:** hard
**Description:** A secure bank account app with PIN login, transactions and file storage.
**Features:**
- Login with a PIN (3 attempts max)
- Check balance
- Deposit money
- Withdraw money (prevent overdraft)
- Save balance to a file between sessions
- Transaction history display
**Constructs:** variables, selection, while loops, functions, file handling, casting, procedures

## Project 5: Restaurant Ordering System
**Difficulty:** medium
**Description:** A menu-driven ordering system that builds an order and prints a receipt.
**Features:**
- Display a menu with item names and prices
- User adds items to their order
- Running total shown after each item
- Option to remove items
- Print final receipt with all items and total
- Save receipt to a file
**Constructs:** lists, procedures, loops, calculations, file handling, concatenation

## Project 6: Quiz Game
**Difficulty:** medium
**Description:** A multiple choice quiz with scoring, random question order and a high score file.
**Features:**
- At least 10 multiple choice questions (stored in a list)
- Random question order each game
- Score tracked throughout
- Correct/wrong feedback after each question
- Save high score to a file
- Easy and hard difficulty levels
**Constructs:** lists, random, file handling, loops, functions, strings, selection

## Project 7: Library Management System
**Difficulty:** hard
**Description:** A book management system to add, borrow, return and search a library catalogue.
**Features:**
- Display all books
- Add new books
- Borrow a book (mark as unavailable)
- Return a book (mark as available)
- Search by title or author
- Save catalogue to a file
**Constructs:** lists, file handling, functions, string slicing, loops, selection

## Project 8: Student Grade Tracker
**Difficulty:** medium
**Description:** Enter student marks, calculate statistics and save reports to a file.
**Features:**
- Enter names and marks for multiple students
- Calculate class average
- Find highest and lowest mark
- Display pass/fail for each student (pass mark 50)
- Save full report to a file
- Display sorted results (highest to lowest)
**Constructs:** lists, loops, functions, file handling, casting, selection

## Project 9: Vending Machine Simulator
**Difficulty:** medium
**Description:** A coin-operated vending machine with stock management and change calculation.
**Features:**
- Display items and prices
- User selects an item
- User inserts coins (loop until enough inserted)
- Calculate and display change
- Reduce stock after each purchase
- Display "sold out" when stock reaches zero
**Constructs:** calculations, selection, loops, functions, lists, casting

## Project 10: Hotel Booking System
**Difficulty:** hard
**Description:** A room booking system with availability checking, customer records and cost calculation.
**Features:**
- Display available rooms (stored in a list)
- Book a room (enter customer name and nights)
- Calculate total cost
- Mark room as unavailable after booking
- View all current bookings
- Save bookings to a file
- Cancel a booking
**Constructs:** lists, functions, file handling, validation, selection, loops, casting
